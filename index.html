Supabase 설정 방법

1. supabase.com → 프로젝트 선택
2. 왼쪽 메뉴 Settings → API 클릭
3. Project URL 복사 → 위 URL란 붙여넣기
4. anon public 키 복사 → 위 Key란 붙여넣기
5. 저장 버튼 클릭

6. SQL Editor에서 supabase_setup.sql 파일 내용 실행
